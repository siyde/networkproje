b) 
genel yapı(TTT şu an yok sadece kart olarak gözüküyor):
/home/ubuntu/pictionary/
│
├── server.py                      # Tüm oyunların (Pictionary, Codenames, TTT) backend'i
│
└── /var/www/gamesite/games/
     ├── pictionary/               # Pictionary HTML istemcisi
     ├── codenames/                # Codenames HTML istemcisi
     └── ttt/                      # TicTacToe HTML istemcisi

c) 
dosya düzenleme sudo nano ile:
sever.py için
cd ~/pictionary
sudo nano server.py

codenames in html dosyası için
sudo nano /var/www/gamesite/games/codenames/index.html

pictionary html dosyası için
sudo nano /var/www/gamesite/games/pictionary/index.html

d) 
!!!!!!!!!!!!!!değişikliklerden sonra UNUTMAYIN!!!!!!!!!!!!!

sadece index.html leri değiştirdiyseniz bunu yaz:
sudo nginx -t && sudo systemctl reload nginx

sudo systemctl restart pictionary
sudo systemctl status pictionary #----> burada active(running) yazmalı
